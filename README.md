## 云南省干部在线学习学院 必修课学习脚本
运行后不需要手动切换视频进行观看，自动输入验证码进行验证
https://www.ynsgbzx.cn/

### 使用说明
需要首先安装aibote
下载地址：http://www.aibote.net/download.html

python版本为：3.10.13

安装依赖
```bash
pip install -r requrements.txt
```

修改web_bot.py文件中 15行和16行的 用户名和密码，改为自己的
```python
USERNAME = "用户名"
PASSWORD = "密码"
```

需要安装官方版本的chrome浏览器

然后启动运行
```bash
python web_bot.py
```
